# Guark Vue Template
Guark Vue.js template.

![Guark Vue Template](https://i.imgur.com/4LDx4OT.png)

## Install

Create new empty directory and cd to it, and run:
```bash
guark new --template vue
```

## Run dev app.

Run dev app with vue hot reload:
```bash
guark dev
```

## Build

Build you app bundle.
```bash
guark build
```
